# VISUAL MEMORY - Memory Game in MS Excel


## PROJECT DESCRIPTION:
Visual Memory is a logical memory game written in **VBA**.
The player's task is to memorize a brief pattern of white tiles
appearing on the board and then recreate that sequence without error.
With each subsequent Level, the difficulty increases: the number
of TilesToFind grows, and the BoardSize scales dynamically.

## HOW TO RUN THE GAME (IMPORTANT):
1. Open the game file (the file must have the `.xlsm` extension).
2. Upon the first launch, Excel will likely display a yellow warning bar (below the main menu).
3. Click the "Enable Content" button to allow macros to run.
Without this step, the game logic will not function!
4. The game will initialize automatically, running the DrawBoard
procedure and preparing the board for the first level.

## SYSTEM REQUIREMENTS:
- Microsoft Excel (2010 or newer recommended due to RGB color support).
- Macros enabled.
- Operating System: Windows (or macOS with VBA environment support).
